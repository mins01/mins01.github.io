- mysql &gt; source /temp/test.sql
- 서버 속의 /temp/test.sql 을 mysql 로 실행한다.
- mysql&gt; \. /temp/test.sql
- 이처럼 줄일 수 있음


- $ mysql -uUSER -pPASS &lt; /temp/test.sql
- 서버 속의 /temp/test.sql 을 mysql 로 실행한다.

- \! {명령어}
- \! ls , \! sar  등등 시스템의 명령어를 실행할 수 있다.
